# Kuvaus

Geneeriset uutiset on web-sovellus, jossa voi lukea uutisia. Etusivulla näkyy aina viisi uusinta uutista kuvineen. Uutisia voi myös listata kategorian perusteella, johon löytyy linkit navigaatiopalkista sivun yläosasta. Etusivulta pääsee myös katselemaan julkaisuajankohdan perusteella listattuja uutisia tai katsomislukujen perusteella suosituimpia uutisia. 

Sovellus sisältää myös hallintapaneelin, jonka avulla uusia uutisia voi julkaista tai olemassaolevia editoida. Hallintapaneeli näkyy vain kirjautuneille käyttäjille, joilla on admin-rooli.
