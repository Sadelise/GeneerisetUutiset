# Käyttötapaukset

## Käyttäjä selaa uutisia
Käyttäjä: kuka tahansa
Tavoite: lukea uutisia
Laukaisija: Halu tietää maailman menosta
Käyttötapauksen kulku:
1. Käyttäjä siirtyy sivulle
2. Käyttäjä lukee uusimpia otsikoita etusivulta
3. Käyttäjä klikkaa etusivulta uutisen auki
4. Käyttäjä lukee uutisen sisällön

## Käyttäjä haluaa lukea tietyn kategorian uutisia
Käyttäjä: kuka tahansa
Tavoite: lukea tietynlaisia uutisia
Laukaisija: Halu tietää maailman menosta mutta vain tietystä kategoriasta
Käyttötapauksen kulku:
1. Käyttäjä siirtyy sivulle
2. Käyttäjä klikkaa haluamaansa kategoriaa navigaatiopalkista
3. Käyttäjä lukee otsikoita kategoriasivulta
4. Käyttäjä klikkaa uutisen auki
5. Käyttäjä lukee uutisen sisällön

## Käyttäjä haluaa lukea uusimpia uutisia
Käyttäjä: kuka tahansa
Tavoite: lukea uutisia
Laukaisija: Halu tietää nykymaailman menosta
Käyttötapauksen kulku:
1. Käyttäjä siirtyy sivulle
2. Käyttäjä klikkaa uusimmat uutiset linkkiä etusivulla
3. Käyttäjä lukee otsikoita
4. Käyttäjä klikkaa uutisen auki
5. Käyttäjä lukee uutisen sisällön

## Käyttäjä lisää uutisen
Käyttäjä: kuka tahansa
Tavoite: julkaista uutinen
Laukaisija: Halu lisätä tietoisuutta maailman menosta
Esiehto: Jotakin uutiskynnyksen ylittävää on tapahtunut
Jälkiehto: Uutinen näkyy kaikille käyttäjille
Käyttötapauksen kulku:
1. Käyttäjä siirtyy sivulle
2. Käyttäjä klikkaa hallintapaneelin auki navigaatiopalkista
3. Käyttäjä syöttää uuden uutisen tiedot avautuneeseen lomakkeeseen
4. Käyttäjä painaa "julkaise"-nappulaa

## Käyttäjä muokkaa uutista
Käyttäjä: kuka tahansa
Tavoite: muokata uutista
Laukaisija: tarve korjata uutisten sisältöä
Esiehto: uutinen on julkaistu
Jälkiehto: uutisen sisältö on muuttunut
Käyttötapauksen kulku:
1. Käyttäjä siirtyy sivulle
2. Käyttäjä klikkaa hallintapaneelin auki navigaatiopalkista
3. Käyttäjä selaa uutisia ja klikkaa sitä uutista, jota haluaa muokata
4. Käyttäjä korjaa syötteet lomakkeessa, johon on automaattisesti tuotu halutun uutisen tiedot 
5. Käyttäjä painaa "muokkaa"-nappulaa
